Temperature Analyzer

Overview

A compact Arduino‑based temperature analyzer built to capture, display, and log temperature data for quick experiments and classroom demos. The system combines a digital sensor, a simple user interface, and a 3D‑printed enclosure for durability.

Arduino Digital sensor Data logging 3D‑printed enclosure

Design Engineering Process

1) Problem Framing

Reliable, low‑cost tool to measure and visualize temperature changes for labs and quick tests.

2) Requirements

±0.5–1.0 °C accuracy
1–2 s refresh
CSV logging
Battery or USB power

3) Concepts & Sketches

Explored handheld vs. benchtop layouts; sensor placement and airflow; display and button positions.

4) CAD & BOM

Modeled enclosure and standoffs; selected sensor, display, buttons, and connectors for quick assembly.

5) Prototype Build

Wired Arduino + sensor on perfboard; printed enclosure; brought up firmware and UI.

6) Test & Iterate

Calibration against a reference thermometer; noise filtering and sampling‑rate tuning.

7) Final Build

Clean wiring, labeled headers, tightened tolerances; improved USB strain relief.

8) Next Steps

Multi‑sensor support, humidity option, and web dashboard via serial/ble.

Hardware & CAD

  • Sensor: digital temperature sensor (drop‑in footprint for thermistor if needed)
  • MCU: Arduino‑class dev board
  • UI: small display + single button (start/mark)
  • Enclosure: 3D‑printed body with threaded inserts

Results

The analyzer achieves stable readings within the target accuracy band and logs directly to CSV for quick analysis. The enclosure protects wiring and makes the device classroom‑friendly.
